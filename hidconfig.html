Overview
Assignments (add / edit / remove)

Each row is one input. Pick its Input and Output; the layer columns show what fires on each layer (click to change). Group is optional. Ō£Ģ removes the row.

Devices

Every input device is an entry here -- there are no built-ins. A device's Baseline maps its physical keys to your standard layout; all Custom Mappings (layers, chords, combos, macros) are shared and apply identically on every device. Pin = any unique part of the device name shown by the Bluetooth tab or RAW log; one entry covers all of a device's interfaces (keyboard + trackpad).

Then Ō¼å Save to device; the engine picks up new devices on service restart.

Include a trailing space if you want it to fire only after a space (e.g. dt ). These save with the top Ō¼å Save to device, not "Save expansions".
